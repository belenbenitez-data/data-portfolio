\# Retail Audit Dashboard (Power BI)



\## 🧠 Project Overview

This project presents an interactive Power BI dashboard developed for a retail audit analysis.  

The dashboard was built on top of a SQL data model that processes point-of-sale (PDV) audit data, including product availability, pricing accuracy, and store execution compliance.



\---



\## 🎯 Business Objective

The main goal of this project is to transform raw SQL-based audit data into actionable business insights for retail performance monitoring.



It helps stakeholders answer questions such as:

\- Are products available on the shelf across all stores?

\- Are there pricing inconsistencies between reported and reference values?

\- How well are stores complying with execution standards?



\---



\## 🛠️ Tools \& Technologies

\- Power BI

\- SQL (data transformation layer)

\- Data modeling (star schema approach)

\- DAX measures (if applicable)



\---



\## 📊 Key Metrics / KPIs

\- Product availability rate (% coverage)

\- Out-of-stock detection by category and store

\- Pricing deviation analysis

\- Store execution compliance (FEFO, expired products, shelf conditions)



\---



\## 🔗 Data Source

The dataset is based on a SQL analysis project using the following tables:

\- dim\_pdv

\- dim\_productos

\- hechos\_encuestas
